
# Roulette Statistical Engine

Este projeto é uma ferramenta de análise estatística para roleta (Europeia), desenvolvida para identificar desvios de frequência em tempo real utilizando o conceito de Z-Score.

## Funcionalidades
- **Input em Tempo Real**: Interface reativa para inserir números conforme o sorteio.
- **Cálculo de Desvio (Z-Score)**: Identifica quais números estão operando estatisticamente abaixo ou acima da média.
- **Persistência Local**: Salva o progresso no navegador automaticamente.

## Arquitetura Sugerida
- `index.html`: Contém a interface, o estilo e a lógica de processamento estatístico em JS.
- `README.md`: Documentação do projeto.
